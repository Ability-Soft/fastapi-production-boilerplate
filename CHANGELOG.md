# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-05-13

### Added
- Initial release of the production-ready FastAPI boilerplate.
- JWT Authentication with refresh token support.
- Role-Based Access Control (RBAC).
- PostgreSQL integration with async SQLAlchemy.
- Alembic for database migrations.
- Redis support for caching.
- Multi-stage Docker build for production.
- Comprehensive test suite with Pytest.
- Automated code quality with Ruff and Mypy.
- GitHub Actions CI/CD workflow.
- Pre-commit hooks for local development.
- Professional GitHub documentation (Contributing, Security, License, Templates).

### Fixed
- Improved `.dockerignore` to reduce image size.
- Standardized error handling responses.
