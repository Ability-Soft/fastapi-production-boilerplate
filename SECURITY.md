# Security Policy

## Supported Versions

We only provide security updates for the latest version of this boilerplate.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |

## Reporting a Vulnerability

AbilitySoft takes the security of our software seriously. If you believe you have found a security vulnerability, please do NOT open a public issue.

Instead, please report it via email to **support@abilitysoft.net**.

### What to include:
- A detailed description of the vulnerability.
- Steps to reproduce (Proof of Concept).
- Potential impact.

### Our Commitment:
- We will acknowledge receipt of your report within 48 hours.
- We will provide an estimated timeline for a fix.
- We will credit you for the discovery (unless you prefer to remain anonymous).

Thank you for helping keep AbilitySoft and our users safe!
